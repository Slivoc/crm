Instructions:
  • Export Purchase Orders from your ERP system as CSV (comma-delimited)
  • The system will automatically create suppliers if they don't exist
  • Multiple lines for the same PO number will be grouped together
  • Part numbers will be added to the system automatically
  • Expected columns:
    • PO Number - Purchase order reference
    • Supplier Code - Supplier identifier or name
    • Date Issued - PO issue date
    • Line Number - Line item number
    • Part Number - Part/product number
    • Quantity - Order quantity
    • Unit Price - Price per unit
    • Ship Date - Expected ship date
    • Currency - Currency code (e.g., USD, GBP)
    • Payment Terms - Payment terms (optional)
    • Incoterms - Delivery terms (optional)
Important Notes:
  • Column names must match exactly as shown above (case-sensitive)
  • If a column is not present in your CSV, update the COLUMN_MAPPINGS in the Python code
  • Dates can be in various formats: YYYY-MM-DD, DD/MM/YYYY, MM/DD/YYYY
  • Duplicate PO numbers will be skipped - only new POs will be created
Sample CSV Format
PO Number,Supplier Code,Date Issued,Line Number,Part Number,Quantity,Unit Price,Ship Date,Currency,Payment Terms,Incoterms
PO-2024-001,ACME CORP,2024-11-01,1,ABC-123,10,150.00,2024-11-15,USD,Net 30,FOB
PO-2024-001,ACME CORP,2024-11-01,2,XYZ-456,5,200.00,2024-11-15,USD,Net 30,FOB
PO-2024-002,WIDGET LTD,2024-11-02,1,PART-789,20,75.50,2024-11-20,GBP,Net 60,CIF
